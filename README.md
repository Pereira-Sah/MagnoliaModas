# 🛒 Magnolia Modas (Frontend)

Este é o aplicativo mobile do nosso projeto, desenvolvido com **React Native** e **Expo**.

## 🚀 Como rodar o app

Abra o terminal na pasta do projeto e siga os passos:

### 1. Instalar as dependências
```bash
npm install
```
### 2. Rodar o projeto
```bash
npx expo start
```
